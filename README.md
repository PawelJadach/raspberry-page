## Run Locally

Clone the project

```bash
  git clone https://github.com/PawelJadach/raspberry-page.git
```

Go to the project directory

```bash
  cd raspberry-page
```

Install dependencies

```bash
  npm install
```

Start the dev server

```bash
  npm run dev
```
